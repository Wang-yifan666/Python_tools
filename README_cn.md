# Python Scripts

这是一个用于整理我平时编写的 Python 脚本的仓库，方便日常学习和使用。

目前仓库包括：

- `rename_yolov5_pairs.py`：用于批量重命名 YOLOv5 数据集中的图片-标签配对文件
- `rename_files_in_order.py`：用于按文件名顺序批量重命名指定文件夹中的文件
- `yolo_visualizer.py`：用于将 YOLO 格式的标注框绘制到图片上，生成可视化结果

更详细的使用说明请查看：

- `rename_yolov5_pairs.py`

- `rename_files_in_order.py`
  - 中文文档：`docs/cn/rename_files_in_order.md`
  - English documentation: `docs/en/rename_files_in_order.md`

- `yolo_visualizer.py`
  - 中文文档：`docs/cn/yolo_visualizer.md`
